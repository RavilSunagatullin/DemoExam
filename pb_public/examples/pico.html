Pico.css: расширенный пример

Страница показывает паттерны из официальной документации Pico и одновременно подгружает данные коллекции example из PocketBase.

Pico работает в первую очередь через семантический HTML: заголовки, формы, таблицы и кнопки стилизуются без дополнительных утилитных классов.

Быстрый старт из доки

Варианты подключения: локальный файл, CDN или пакетный менеджер. В проекте используется локальный файл из /assets/pico.min.css. 

<!-- Локально в проекте -->
<link rel="stylesheet" href="/assets/pico.min.css" />

<!-- CDN -->
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@picocss/pico@2/css/pico.min.css"
/>

Аккордеон

Почему семантический HTML?

Меньше кастомного CSS, быстрее верстка и понятнее код.

Как подключить Pico в этом проекте?

Через тег <link rel="stylesheet" href="/assets/pico.min.css" /> в секции head.

Как здесь используется PocketBase?

Страница вызывает list("example") и отображает результат в таблице.
