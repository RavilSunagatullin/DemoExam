## Использование

```bash
pocketbase.exe [команда]
```

## Доступные команды

- `migrate` — выполняет скрипты миграции БД приложения
- `serve` — запускает веб-сервер (по умолчанию `127.0.0.1:8090`, если домен не указан)
- `superuser` — управление суперпользователями
- `update` — автоматически обновляет текущий исполняемый файл приложения до последней доступной версии

## Быстрый старт

```powershell
.\pocketbase.exe serve --http=127.0.0.1:8090
```

Откройте:

- Главная: `http://127.0.0.1:8090/`
- Админка: `http://127.0.0.1:8090/_/`
